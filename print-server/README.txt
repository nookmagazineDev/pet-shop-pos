Print Server ทำงานยังไง?
────────────────────────────────────────────────────────────────

  ┌──────────────┐   WiFi/LAN    ┌─────────────────┐    TCP     ┌───────────────┐
  │  มือถือ/คอม  │ ────────────► │  Print Server   │ ─────────► │  เครื่องพิมพ์ │
  │  (เว็บแอป)   │               │  (คอมในร้าน)    │  Port 9100 │  Thermal      │
  └──────────────┘               │  Port 3001      │            └───────────────┘
                                 └─────────────────┘

  • Print Server รันบนคอมในร้านตัวหนึ่ง (ต้องเปิดทิ้งไว้)
  • ทุกอุปกรณ์ที่อยู่ WiFi เดียวกัน เรียกพิมพ์ผ่าน IP ได้เลย
  • ไม่ต้องติดตั้ง Driver เครื่องพิมพ์ เลย

────────────────────────────────────────────────────────────────
  ความต้องการของระบบ
────────────────────────────────────────────────────────────────

  ✅  Node.js เวอร์ชัน 14 ขึ้นไป
      (ติดตั้งจาก โฟลเดอร์ setup/ ที่แนบมาให้)

  ✅  เครื่องพิมพ์ Thermal ที่รองรับ ESC/POS ผ่าน TCP Port 9100
      เช่น Epson TM-T82, Gprinter, Xprinter, Bixolon, Star

  ✅  เครื่องพิมพ์และคอมที่รัน Print Server
      ต้องต่อ WiFi / LAN เดียวกัน

────────────────────────────────────────────────────────────────
  STEP 1 — ติดตั้ง Node.js (ทำครั้งแรกครั้งเดียว)
────────────────────────────────────────────────────────────────

  ไปที่โฟลเดอร์:  setup/
  ดับเบิลคลิก:    ติดตั้ง-NodeJS.bat
  รอจนเสร็จ แล้ว Restart คอมพิวเตอร์

  ดูคู่มือเพิ่มเติมในไฟล์:  setup/README.txt

────────────────────────────────────────────────────────────────
  STEP 2 — เปิด Print Server (ทำทุกครั้งก่อนใช้งาน)
────────────────────────────────────────────────────────────────

  ไปที่โฟลเดอร์หลัก:  pet-shop-pos/
  ดับเบิลคลิก:         เริ่มระบบปริ้นเตอร์.bat

  หน้าต่างดำจะขึ้นแบบนี้:

  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  ✅  Print Server เริ่มทำงานแล้ว!                       │
  │      Local  : http://localhost:3001                      │
  │      Network: http://192.168.1.5:3001  ← IP ในร้าน      │
  │                                                          │
  └──────────────────────────────────────────────────────────┘

  ⚠  ห้ามปิดหน้าต่างนี้ระหว่างใช้งาน
  📌  จด IP บรรทัด Network ไว้ใช้ STEP 4

────────────────────────────────────────────────────────────────
  STEP 3 — หา IP เครื่องพิมพ์
────────────────────────────────────────────────────────────────

  วิธีที่ 1 — พิมพ์ใบ Config จากตัวเครื่อง
  ┌──────────────────────────────────────────┐
  │  กดปุ่ม [FEED] ค้างไว้ 3 วินาที         │
  │  เครื่องพิมพ์จะพิมพ์ใบรายงาน            │
  │  มองหาบรรทัด "IP Address" หรือ "IPv4"   │
  └──────────────────────────────────────────┘

  วิธีที่ 2 — ดูจาก Router Admin
    เปิด browser → พิมพ์  192.168.1.1  หรือ  192.168.0.1
    ดูในหน้า "Connected Devices" หรือ "DHCP Clients"

  วิธีที่ 3 — พิมพ์ Self-Test
    กดปุ่ม [FEED] ค้าง แล้วเปิดสวิตช์เครื่องพิมพ์พร้อมกัน
    IP จะพิมพ์ออกมาในใบ Self-Test

────────────────────────────────────────────────────────────────
  STEP 4 — ตั้งค่าในเว็บแอป (ทำครั้งแรกหรือเมื่อเปลี่ยน WiFi)
────────────────────────────────────────────────────────────────

  เปิดเว็บแอป PetShop POS
  ไปที่เมนู  ตั้งค่าปริ้นเตอร์

  กรอกข้อมูล:

  ┌────────────────────────────────────────────────────────┐
  │  IP เครื่องพิมพ์                                      │
  │  → ใส่ IP ที่ได้จาก STEP 3  เช่น  192.168.1.100      │
  │                                                        │
  │  เปิด Toggle  "เซิร์ฟเวอร์ปริ้นจิ๋ว (Direct Print)"  │
  │  → เปิดเพื่อพิมพ์โดยไม่มี dialog เด้ง                │
  │                                                        │
  │  Print Server URL                                      │
  │  ┌──────────────────────────────────────────────────┐ │
  │  │ ใช้จากคอมที่รัน .bat  → http://localhost:3001    │ │
  │  │ ใช้จากมือถือ/คอมอื่น  → http://192.168.1.5:3001 │ │
  │  │   (IP ที่เห็นใน cmd บรรทัด Network จาก STEP 2)  │ │
  │  └──────────────────────────────────────────────────┘ │
  └────────────────────────────────────────────────────────┘

  กด "บันทึก"
  กด "ทดสอบพิมพ์" → ได้ยินเสียงตัดกระดาษ = สำเร็จ ✅

────────────────────────────────────────────────────────────────
  แก้ปัญหาเบื้องต้น
────────────────────────────────────────────────────────────────

  ❌  "เชื่อมต่อ Print Server ไม่ได้"
      → ตรวจว่าหน้าต่าง .bat ยังเปิดอยู่ไหม
      → ตรวจว่า Print Server URL ตรงกับ IP ใน cmd

  ❌  "'node' ไม่ใช่คำสั่งที่รู้จัก"
      → ยังไม่ได้ติดตั้ง Node.js หรือยังไม่ได้ Restart
      → ทำ STEP 1 ใหม่

  ❌  "npm install ไม่สำเร็จ"
      → ลบโฟลเดอร์ node_modules แล้วลองใหม่
      → หรือรัน:  npm install --legacy-peer-deps

  ❌  Print Server เริ่มได้ แต่พิมพ์ไม่ออก
      → ตรวจสอบ IP เครื่องพิมพ์ใน STEP 3
      → เครื่องพิมพ์ต้องเปิดอยู่และอยู่ WiFi เดียวกัน
      → ทดสอบ:  ping 192.168.1.100  ใน cmd

  ❌  ตัวอักษรภาษาไทยเป็นกล่องสี่เหลี่ยม
      → ตั้งค่า Code Page เครื่องพิมพ์เป็น  TIS-620 / CP874
      → ดูคู่มือเครื่องพิมพ์ส่วน "Character Set" หรือ "Code Page"

  ❌  มือถือเรียก Print Server ไม่ได้
      → ตรวจ Firewall ของ Windows:
        Control Panel → Windows Defender Firewall
        → Allow an app through firewall → เพิ่ม Node.js
        หรือเปิด port 3001 ใน Inbound Rules
