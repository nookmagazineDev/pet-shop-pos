================================================================
  โฟลเดอร์ setup — ติดตั้งก่อนใช้งานครั้งแรก
================================================================

ไฟล์ในโฟลเดอร์นี้:

  node-v24.15.0-x64.msi   — ตัวติดตั้ง Node.js (Windows 64-bit)
  ติดตั้ง-NodeJS.bat       — สคริปต์ติดตั้งอัตโนมัติ

================================================================
  ขั้นตอนติดตั้ง (ทำครั้งแรกครั้งเดียว)
================================================================

1. ดับเบิลคลิก  "ติดตั้ง-NodeJS.bat"
   → ระบบจะติดตั้ง Node.js ให้อัตโนมัติ
   → ถ้าติดตั้งแล้ว จะแจ้งให้ทราบโดยไม่ติดตั้งซ้ำ

2. Restart คอมพิวเตอร์

3. กลับไปที่โฟลเดอร์หลัก
   ดับเบิลคลิก  "เริ่มระบบปริ้นเตอร์.bat"

================================================================
  หมายเหตุ
================================================================

- Node.js เวอร์ชัน 24.15.0 LTS (Windows x64)
- ต้องใช้ Windows 10 ขึ้นไป
- ต้องมีสิทธิ์ Administrator ในการติดตั้ง
