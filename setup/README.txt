ทำไมต้องติดตั้ง?
  Print Server ต้องใช้ Node.js เพื่อทำงานได้
  ติดตั้งครั้งเดียว ไม่ต้องทำซ้ำอีก

────────────────────────────────────────────────────────────────
  โฟลเดอร์นี้มีอะไรบ้าง
────────────────────────────────────────────────────────────────

  setup/
  ├── node-v24.15.0-x64.msi    ← ตัวติดตั้ง Node.js (Windows 64-bit)
  ├── install-nodejs.bat        ← ดับเบิลคลิกเพื่อติดตั้งอัตโนมัติ
  └── README.txt                ← คู่มือนี้

────────────────────────────────────────────────────────────────
  ขั้นตอนติดตั้ง
────────────────────────────────────────────────────────────────

  STEP 1 — ดับเบิลคลิก "install-nodejs.bat"
  ┌─────────────────────────────────────────────┐
  │  install-nodejs.bat                         │
  │  ↓                                          │
  │  หน้าต่างดำขึ้นมา → ระบบติดตั้ง Node.js    │
  │  รอจนขึ้นว่า "installed successfully!"      │
  └─────────────────────────────────────────────┘

  หมายเหตุ:
    - ถ้าติดตั้งอยู่แล้ว → จะแจ้งให้ทราบ ไม่ต้องทำซ้ำ
    - ถ้า Windows ถามให้ยืนยัน (UAC) → กด "Yes"

  ──────────────────────────────────────────────

  STEP 2 — Restart คอมพิวเตอร์

  สำคัญ: ต้อง Restart ก่อน ไม่งั้น Node.js ใช้ไม่ได้

  ──────────────────────────────────────────────

  STEP 3 — กลับไปโฟลเดอร์หลัก
           ดับเบิลคลิก "start-print-server.bat"

  ┌─────────────────────────────────────────────┐
  │  pet-shop-pos/                              │
  │  └── start-print-server.bat  ← ดับเบิลคลิก │
  └─────────────────────────────────────────────┘

────────────────────────────────────────────────────────────────
  ปัญหาที่พบบ่อย
────────────────────────────────────────────────────────────────

  ดับเบิลคลิก .bat แล้วหน้าต่างดำหายเร็วมาก
  → คลิกขวา → "Run as administrator"

  ขึ้น "File not found: node-v24.15.0-x64.msi"
  → ตรวจสอบว่า .msi อยู่ในโฟลเดอร์เดียวกับ .bat

  ติดตั้งแล้ว แต่เปิด .bat ที่โฟลเดอร์หลักยังขึ้น error
  → ยังไม่ได้ Restart คอมพิวเตอร์ → Restart ก่อน
